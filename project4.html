Mausam

a javascript weather app

Introducing my latest project - a full-stack weather app powered by Node.js, Express.js, and EJS. Utilizing the Open Weather API, this app delivers accurate weather data straight from the backend server, crafted with Express.js. Stay on top of the weather conditions with ease, all in one convenient and user-friendly app.

The technology stack for this application includes Html, CSS on the client-side and node.js, express.js and ejs on the backend side. These technologies were carefully chosen to provide a seamless user experience, efficient data management, and real-time communication capabilities.
