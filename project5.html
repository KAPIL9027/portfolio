AI Image Generator

full stack image generator app

Introducing my latest GitHub project - a powerful full stack web application that utilizes OpenAI's API to generate high quality images based on text descriptions provided by the user. With this cutting-edge app, you can create stunning images with ease, simply by describing what you envision. The user-friendly interface is designed to operate like MidJourney AI, making the process effortless and seamless. But that's not all - my application also features a search function, allowing users to easily find specific images as needed.

Built using the MERN stack, this app leverages React.js for the frontend, and Node.js, Express.js, MongoDB, and Cloudinary for the backend. With this comprehensive and robust stack, you can rest assured that your experience will be smooth and enjoyable from start to finish.
